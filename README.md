# Comic Cralwer

## Requirements
- python3
- chrome web driver
  - from https://chromedriver.chromium.org/downloads
- modules in requirements.txt
- headers.json, setting.conf, website.txt
```
pip3 install -r requirements.txt
```
---
## website.txt

```
1. how many
2. website address
3. next button, prev button(just write 'next', 'prev')
4. comic name
5. comic site
```

eg.
```
1
www.example.com/1
next
comic_name
haha
```
---
## Run
```
python main.py
```
